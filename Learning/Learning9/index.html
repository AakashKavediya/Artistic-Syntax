Learning: HTML Image

HTML images are just a kind of image. There is nothing much to talk about it. The HTML support all type of image and it also support gif.

The image tag can help us to add more effect with the help of CSS. You can blur the image, you can change the color of it. You can also change the height width of the image. With the help of CSS you can also change the shape of the image and can also add a layer of color on it.
